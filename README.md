# ROS 2 Mobile Robot Modeling and Simulation

ROS 2 Humble mobile robot simulation project featuring a custom URDF/Xacro robot, Gazebo world, RViz visualization, ROS 2 Control integration, and a Turtlesim bridge.

## Packages Included
* **robot_description**: URDF/Xacro modeling and configuration files.
* **edges_bringup**: Launch files to start the simulation.
* **gazebo_turtle_bridge**: Bridge node for control logic.
